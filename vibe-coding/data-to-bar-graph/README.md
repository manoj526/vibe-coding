# Data to Bar Graph

This project is designed to convert data into a bar graph using Python. It utilizes various libraries for data handling and visualization.

## Project Structure

```
data-to-bar-graph
├── src
│   ├── main.py          # Entry point of the application
│   ├── utils
│   │   └── graph_utils.py # Utility functions for graph creation
├── requirements.txt     # List of dependencies
└── README.md            # Project documentation
```

## Installation

To get started, clone the repository and navigate to the project directory:

```bash
git clone <repository-url>
cd data-to-bar-graph
```

Next, install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

To run the application, execute the following command:

```bash
python src/main.py
```

## Example

You can modify the `src/main.py` file to input your own dataset and generate a bar graph. The `create_bar_graph(data)` function from `graph_utils.py` will be used to visualize the data.
